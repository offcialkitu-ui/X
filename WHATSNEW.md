## What's New (v6.3.2)

### 🆕 Redesigned Update Experience
- **Brand New Update UI** — A modern, dark-themed update screen with clear version titles, release metadata, and scrollable changelogs.
- **Live Download Progress** — Track your update downloads in real-time with a smooth horizontal progress bar and dynamic percentage indicator.
- **In-App Installation** — New confirmation dialog for a seamless transition from downloading to updating.

### 🎵 Upstream Engine Enhancements
- **Age-Restricted Playback Fix** — Integrated JioSaavn fallback so blocked or age-restricted YouTube tracks now stream automatically.
- **Turbo Downloads** — Implemented fast 5MB chunked downloads to bypass YouTube network speed throttling.
- **Offline Data Saver** — Optimized background tasks so playing downloaded music no longer wastes mobile data.

### 🎨 Visuals & Sharing
- **Cosmic Dust Animation** — Added a new minimalist "Cosmic Dust" background style for the player and Ambient Mode.
- **Premium Story Share** — New high-end Instagram Story templates with dynamic glass effects and mood pulse rings.

### 🔧 Fixes & Internal
- Fixed Liquid Glass adaptive text contrast.
- Resolved multiple resource conflicts and preference key inconsistencies.
- Improved build stability for Universal FOSS variants.
